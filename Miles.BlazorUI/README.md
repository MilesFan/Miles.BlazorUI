# Blazor Image Viewer
A simple Blazor component for viewing images with zoom and pan functionality.
## Features
- Zoom In/Out with Mouse Wheel
- Pan by Dragging the Image

## Usage
```
<ImageViewer
    ImageUrl="/img/bingwallpaper2.jpg"
    Style="width:800px; height:600px; border: 1px solid gray; background-color:#00000060;" />
```
